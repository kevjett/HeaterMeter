Stash (archive) database

The current log file is lost when LinkMeter is rebooted. To make your log available for later review or continuation, the information can be "stashed" to storage. Using the stash command copies the current log file at the point it is stashed.

Storage requirements are about 80KB per stash.

Reload this page after stashing to see the file in the stored list.
